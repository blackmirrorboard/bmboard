# 02. ターミナル全コマンドリファレンス

> Black Mirror Board で使える全コマンドの完全リスト。v1.2 時点。

---

## ターミナルの使い方

画面下部の `❯` マークの横にコマンドを入力 → **Enter** で実行。

### 基本操作

| 操作 | 効果 |
|---|---|
| `↑` / `↓` | コマンド履歴を呼び出し |
| Enter | 実行 |
| Esc | 入力キャンセル |
| Tab（PCのみ） | テーマ切替メニュー等 |

### 入力の形式

以下は**すべて同じコマンド**として認識：

```
scatter
$ scatter
$scatter
```

※ 先頭の `$` は飾り。あってもなくても動く。

---

## 1. テーマ切替

| コマンド | 効果 |
|---|---|
| `dark` | 黒背景・白ストローク |
| `light` | 白背景・黒ストローク |
| `gray` | スタジオモード（ペブル地・コバルト青） |

---

## 2. プロジェクト管理（ブラウザ内保存）

| コマンド | 効果 |
|---|---|
| `save <名前>` | 今のキャンバスを IndexedDB に保存 |
| `ls` | 保存済みの一覧を表示 |
| `load <名前>` | 保存したものを復元 |

### 使用例

```
save draft-01     → 今の状態を "draft-01" として保存
ls                → 保存したもの一覧
load draft-01     → "draft-01" を復元
```

ブラウザをリロードしても消えない。ただしブラウザのデータを消すと一緒に消えるので、重要なら `export data` でファイル書き出しを推奨。

---

## 3. 魔法（プリセット）

最初から入っている魔法：

| コマンド | 効果 |
|---|---|
| `$ scatter` | 選択中のオブジェクトをランダムに散らす |
| `$ grid-3x3` | 3×3 の四角グリッドを配置 |
| `$ biwako-blue` | 全ストロークを青 `#0044CC` に染める |
| `$ fetch-img <キーワード> <枚数>` | 画像をキーワード検索して配置 |
| `$ monoclo` | 選択中の画像をモノクロ化 |
| `$ svg` | 選択オブジェクトを編集可能 SVG として書き出し |

### `fetch-img` の詳細

```
fetch-img                  → "biwako" で1枚
fetch-img mountain         → "mountain" で1枚
fetch-img mountain 6       → "mountain" で6枚・グリッド配置
fetch-img tokyo night 9    → "tokyo night" で9枚
```

最大25枚まで。画像は Lorem Flickr から取得（API キー不要）。

### `monoclo` の詳細

1. 画像を選択
2. ターミナルに `monoclo` → Enter
3. 画像がモノクロになる（Cmd+Z で戻せる）

---

## 4. 魔法の管理

| コマンド | 効果 |
|---|---|
| `list` | 登録済み魔法の一覧（preset / user 区別付き） |
| `register <名前> <JSコード>` | 新しい魔法を作成 |
| `alias <新>=<旧>` | 既存の魔法を別名で複製 |
| `forget <名前>` | ユーザー魔法を削除（preset は削除不可） |
| `share <名前>` | その魔法の JSON をクリップボードにコピー |

自作魔法の作り方は → [[04_自作魔法の作り方_BM_API|04. 自作魔法の作り方]]

---

## 5. 書き出し（Export）

| コマンド | 効果 |
|---|---|
| `export` | 選択オブジェクトを透過 PNG で書き出し |
| `$ svg` | 選択オブジェクトを編集可能 SVG で書き出し（Illustrator / Figma 対応） |
| `export magic` | 魔法ピッカー（個別選択）対話モーダル |
| `export data` | データバンドル書き出し（粒度調整）対話モーダル |
| `export-spells` | 自作魔法すべてをファイル化（非対話） |
| `export-spells a b c` | 指定魔法だけファイル化 |
| `export-all` | キャンバス+全魔法をバンドル化（非対話） |
| `share <名前>` | 1つの魔法をクリップボードに |

詳細 → [[07_魔法の書き出しと共有|07. 魔法の書き出しと共有]]

---

## 6. 取り込み（Import）

専用コマンドは無く、以下の経路で自動認識：

- **ターミナルに JSON をペースト** → 自動登録
- **ボードにファイルを D&D** → 自動判別＆登録

詳細 → [[08_魔法の取り込み|08. 魔法の取り込み]]

---

## 7. ターミナル操作・表示

| コマンド | 効果 |
|---|---|
| `clear` / `flush` | ターミナルログをクリア |
| `help` / `?` | クイックリファレンス表示 |
| `usage` / `man` / `docs` | 使い方オーバーレイを開く |
| `objects` | 現在のキャンバス上の全オブジェクト情報を表示 |
| `test-parse <何か>` | デバッグ用：コマンド解析結果を表示 |

## 8. イースターエッグ

| コマンド | 効果 |
|---|---|
