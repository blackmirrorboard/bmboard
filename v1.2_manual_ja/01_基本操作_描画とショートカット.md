# 01. 基本操作 — 描画とショートカット

> マウス・キーボードでの操作方法を網羅。まずはここを覚えれば紙とペン感覚で使える。

---

## ツール一覧

画面下部のツールバーから選ぶか、キーボードショートカットで切替：

| キー | ツール | 用途 |
|---|---|---|
| **S** | セレクト（選択） | オブジェクトを選ぶ・動かす |
| **P** | ペン（鉛筆） | フリーハンドで描く |
| **N** | パスツール | アンカーポイントで曲線を描く |
| **E** | 消しゴム | クリックしたオブジェクトを削除 |
| **C** | 円 | ドラッグで円を描画 |
| **G** | 三角 | ドラッグで三角を描画 |
| **Q** | 四角 | ドラッグで四角を描画 |
| **A** | 矢印 | ドラッグで矢印を描画 |
| **T** | テキスト | クリックして文字入力 |

---

## 線の太さ

| キー | 太さ |
|---|---|
| **1** | 細（1.5px） |
| **2** | 中（2.5px） |
| **3** | 太（4px） |

---

## 選択と編集

### 選択

| 操作 | 効果 |
|---|---|
| クリック | 単一選択 |
| Shift + クリック | 複数選択の追加／解除 |
| ドラッグ（セレクトツール中） | マーキー選択（矩形で囲む） |
| Esc | 全選択解除 |

### 編集

| 操作 | 効果 |
|---|---|
| ドラッグ | 移動 |
| 角のハンドルをドラッグ | リサイズ |
| 上部のハンドル | 回転 |
| **Del** / **Backspace** | 削除 |
| **Cmd+Z** / **Ctrl+Z** | Undo（取り消し） |
| **Cmd+Shift+Z** | Redo（やり直し） |

---

## キャンバス操作

| 操作 | 効果 |
|---|---|
| **Space + ドラッグ** | パン（画面を動かす） |
| マウスホイール | ズーム |
| ピンチアウト／イン（タッチ） | ズーム |
| 2本指ドラッグ（タッチ） | パン |

---

## テーマ切替

ターミナルで以下を打つと画面の色が変わる：

```
dark     → 黒背景・白い線
light    → 白背景・黒い線
gray     → スタジオモード（ペブル地・青アクセント）
```

---

## 画像の扱い

| 操作 | 効果 |
|---|---|
| **I** キー | ローカル PC から画像アップロード |
| ターミナルで `fetch-img` | キーワードで画像検索して配置 |
| 画像をドラッグ | 移動 |
| 角のハンドル | リサイズ |

---

## 書き出し

| 操作 | 効果 |
|---|---|
| **X** キー | 選択オブジェクトを透過 PNG で書き出し |
| ターミナルで `export` | 同上 |
| ターミナルで `export data` | ワークスペース全体をバンドル JSON に |

---

## モバイルでの操作

### タッチ操作

- **タップ** — 選択・描画開始
- **長押し** — 右クリック相当（コンテキストメニュー）
- **2本指タップ** — 選択解除
- **ピンチ** — ズーム
- **2本指ドラッグ** — パン

### モバイルツールバー

画面下にコンパクト版のツールバーが表示。キーボードショートカットは使えないのでツールバーから切替。

---

## よくやる操作の流れ

### ラフスケッチ

```
1. P キー → ペンで描く
2. E キー → 消しゴムで修正
3. X キー → PNG 書き出し
```

### ムードボード作成

```
1. ターミナルで fetch-img mountain 6
  → 6枚の山の画像がグリッド配置される
2. 気に入らない画像をクリックで選択 → Del で削除
3. T キーでテキスト追加 → 「山のムード案」
4. export で PNG 書き出し
```

### 実験的なレイアウト

```
1. Q キーで四角を何個か描く
2. S キーで全選択（Cmd+A）
3. ターミナルで scatter
  → ランダムに散らばる
4. 気に入ったら保存、気に入らなければ Cmd+Z
```
